# Yantra

## Usage

```
Commands:
  yantra.rb help [COMMAND]                                                                     
  yantra.rb merge FILES DESTIONATION -f, --files=one two three -o, --output=OUTPUT             
  yantra.rb produce SOURCE DESTIONATION -i, --input=INPUT -o, --output=OUTPUT -t, --to, --to=TO

ruby yantra.rb produce -i input-words-list.txt -o output.txt -t de # translate from en to de
```
